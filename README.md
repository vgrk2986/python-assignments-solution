# Решение заданий на Python

## Описание проекта
Этот репозиторий содержит решения двух заданий на Python, выполненные в среде Google Colab.

## Содержание
- `task1_middle_letters.py` - Задание 1: Поиск средних букв в слове
- `task2_dating_service.py` - Задание 2: MVP dating-сервиса  
- `main_program.py` - Главная программа, объединяющая оба задания
- `README.md` - Документация

## Задание 1: Поиск средних букв в слове

### Описание
Программа находит среднюю букву (для слов с нечётной длиной) или две средние буквы (для слов с чётной длиной).

### Примеры работы
- Слово "test" → "es"
- Слово "testing" → "t"

## Задание 2: MVP dating-сервиса

### Описание
Программа создаёт "идеальные пары" путём сортировки списков имён по алфавиту.

### Примеры работы
Пример 1:
Идеальные пары:
Alex и Emma
Arthur и Kate
John и Kira
Peter и Liza
Richard и Trisha

Пример 2:
Внимание, кто-то может остаться без пары!

## Запуск программ

### Способ 1: Запуск главной программы
python main_program.py

### Способ 2: Запуск отдельных заданий
python task1_middle_letters.py
python task2_dating_service.py

## Требования
- Python 3.6+

## Чек-лист проверки

### Задание 1
- При word = 'test' выдается 'es'
- При word = 'testing' выдается 't'

### Задание 2  
- Для равных списков выводятся идеальные пары
- Для неравных списков выводится предупреждение
